# typeUserDisabledAttrs
Disabled attributes for specific field types.

## Usage
```javascript
var options = {
  typeUserDisabledAttrs: {
    'checkbox-group': [
      'name',
      'description',
      'required',
      'inline',
      'toggle'
    ]
  }
};
$(container).formBuilder(options);
```
